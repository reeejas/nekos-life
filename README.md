# nekos.life.py
A Pyhton based discord bot for the https://github.com/Nekos-life Api.

# Requirements
- discord@1.7.3
- aiohttp@3.7.4.post0

# Install
```
pip install discord
pip install aoihttp
change example.config.js -> config.js
edit config.js(prefix, token etc.)
run main.py
```
